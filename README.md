# Swar Mahesh Khatav Portfolio

A modern, responsive personal portfolio website for Swar Mahesh Khatav, built to highlight software engineering experience, projects, technical skills, leadership, and contact information for recruiters.

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Lucide React icons

## Setup

```bash
npm install
npm run dev
```

The local development server will print a localhost URL in the terminal.

## Production Build

```bash
npm run build
npm run preview
```

## Deploying to Vercel

Use these settings in Vercel:

- Framework preset: Vite
- Build command: `npm run build`
- Output directory: `dist`
- Install command: `npm install`

You can deploy by importing the GitHub repository into Vercel, or by using the Vercel CLI.
